# Simulador Orcamento freelancer
Disponibiliza o orçamentos de serviços de TI para para quem deseja contratar (desenvolvimento de site, hospedagem, manutenção, etc.). Mostrando o valor total sendo atualizado dinamicamente.

## Desafios encontrados
Este projeto me fez desafiar o nível do meu conhecimento na criação de um site interativo com o usuário, isso só demonstra que tenho um grande caminho pela frente, muitos conhecimentos ainda estão crus e precisam ser aprofundados.
